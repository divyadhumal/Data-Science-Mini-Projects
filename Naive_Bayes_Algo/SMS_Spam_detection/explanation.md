
# 📩 SMS Spam Detection using Naive Bayes

## 🚀 Project Overview
This project is a simple Machine Learning model that classifies SMS messages as **Spam** or **Not Spam (Ham)** using the Naive Bayes algorithm.

It is built using basic NLP techniques and is beginner-friendly.

---

## 📌 Features
- Text preprocessing and cleaning
- Convert text into numerical form using CountVectorizer
- Train model using Multinomial Naive Bayes
- Predict whether a message is spam or not
- Simple and easy-to-understand implementation

---

## 🛠️ Tech Stack
- Python 🐍
- Pandas
- Scikit-learn
- CountVectorizer (NLP)
- Multinomial Naive Bayes

---

## ⚙️ How It Works
1. Load dataset of SMS messages
2. Split data into training and testing sets
3. Convert text data into numerical vectors
4. Train the Naive Bayes model
5. Make predictions on new messages
